# Klaytn v1.5.1 のリリースノート <a id="release-notes-for-klaytn-v1-5-1"></a>

## Fixes <a id='fixes'></a>
- 不良ブロックの問題を修正しました(#576)
  - 状態トリエ `` ロジックの競合状態を解決しました。
  - trieノードキャッシュの `コミット` と `参照` 中に `逆参照` を防止しました。

## Miscellaneous <a id='miscellaneous'></a>
- RC RPM公開/RCバイナリリンクでPRコメント (#581)
  - RCにRPMを公開するためのジョブを追加しました。 RCバージョンへのインストール/アップグレードは以前よりもはるかに簡単にできます。
  - これからは、すべてのリリースPRのコメントに記載されているリンクからRCバージョンのバイナリをダウンロードできるようになりました。
- Golangバージョンをバージョン1.14.6にアップグレードしました (#582)
  - goコマンド、コンパイラ、リンカー、vet、database/sql、encoding/json、net/http、リフレクト、テストパッケージに修正が含まれています。 (https://golang.org/doc/devel/release.html)

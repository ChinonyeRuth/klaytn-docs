# Release Notes for Klaytn v1.1.1 <a id="release-notes-for-klaytn-v1-1-1"></a>

- Updated package descriptions
- Added copyright message in Service Chain Bridge Contract.
- Removed unused code

# Klaytn v1.1.0 のリリースノート <a id="release-notes-for-klaytn-v1-1-0"></a>

### 新機能 <a id="new-features"></a>
- 特定のブロック番号のスマートコントラクトメソッドの呼び出しをサポート
- ガバナンス評議会メンバーのための複数の投資契約をサポート

### Fixes <a id="fixes"></a>
- Fixed various bugs

### Improvements <a id="improvements"></a>
- コンセンサスメッセージは、ラウンド番号を同期するためにガバナンス評議会メンバーに伝播されます。
- リファクタリングパッケージ (報酬、ガバナンスなど)
- doc.go でパッケージの説明を更新しました
- Added more unit tests

# Release Notes <a id="release-notes"></a>

このバージョンは、セキュリティパッチとトレーサーの機能強化のためのものです。

## New Features
- ブロック範囲内のトランザクションをトレースする新しいAPIを実装しました (#1586 #1590)
    - `debug_traceChain` (`debug_subscribe` with `traceChain` option) は、与えられた範囲のトレース結果を提供するサブスクリプションIDを返します。 この API は websocket 接続経由で呼び出される必要があります
    - `debug_traceBlockByNumberRange` は指定されたブロック数範囲内のすべてのトランザクションをトレースします
- yaml (#1580) を使用して新しいノード設定メソッドを導入しました。

## Improvements
- chaindatafetcherの範囲フェッチのためのメモリ制御ロジックを導入しました (#1583)
- デバッグAPIにおけるtxトレーサーのリファクタリングとメモリ制御の強化 (#1581)

## Fixes
- 送信者が十分な KLAY(#1588) を持っていない場合に発生する見積もったGas APIの失敗を解決しました
- 特別に作成されたp2pメッセージ (https://github.com/klaytn/klaytn/pull/1592/commits/f7270496cb11f248e78d261f289fa4e473da961d) を介してノードパニックを防止

## Miscellaneous
- マイナーメンテナンス (#1566 #1576 #1584 #1585 #1589 #1591)

# Release Notes <a id="release-notes"></a>

Klaytn v1.7.2は、CNがトランザクションを処理し、同時にブロックを生成する際にパニックを引き起こす可能性のある競合状態の問題を修正しました。 このパッチは、CN固有の問題にのみ適用されます。 他のノードタイプ (EN, PN) は Klaytn クライアントをこのバージョンに更新する必要はありません。

## Fix
- txpoolでの競合状態の問題を修正しました (#1071)

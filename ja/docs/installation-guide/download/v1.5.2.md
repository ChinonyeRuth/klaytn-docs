# Klaytn v1.5.2 のリリースノート <a id="release-notes-for-klaytn-v1-5-2"></a>

## 新機能
- ブロックチェーンデータフェッチャーを実装し、リアルタイムでブロックチェーンデータをRDBにプッシュします
  - トークン転送履歴をロードするチェーンデータフェッチャーを実装しました (#586)
  - トランザクションデータを読み込む/取り戻すためのチェーンデータフェッチャーを実装しました (#587)
  - コントラクトデータをロードするチェーンデータフェッチャーを実装しました (#588)

- Service chain Node (#591) での KAS アンカーAPIのサポートの実装
  - サービスチェーン内のブロックは、コマンド オプション (#653) を使用して、KAS アンカーAPI経由で簡単にバオバブ/サイプレスに固定できます。
  - `txpool.allow-local-anchortx` コマンドオプションは、JSON-RPC(#614)経由でアンカートランザクションの送信を許可します。
  - 新しいアンカーデータタイプ(JSON形式)のサポート (#614)

## その他
- 新しいデータベース実装 (DynamoDB) を追加しました。 (#633, #624, #615, #598)
- `overwrite-genesis` フラグを追加し、ジェネシスブロックを既存の状態 (#649) で初期化しました。
- さまざまな種類のキャッシュ実装をサポートするために stateDB キャッシュインターフェースを実装しました (#646)
